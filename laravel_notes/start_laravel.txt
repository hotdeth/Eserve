@this file help you to run laravel on your machine

after install laravel and the composer for your system.
and clone this repo follow this steps:
    ~ goto /Eserve/laravel
    ~ type this command -> `composer install` 
    ~ copy the file .env to /Eserve/laravel 
    ~ copy the file database.sqlite to /Eserve/laravel/database
    ~ type the command `php artisan serve` to test the server
